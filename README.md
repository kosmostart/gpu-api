<h1 align="center">
  gpu-api
</h1>

This is collection of [`wgpu`](https://github.com/gfx-rs/wgpu/) pipelines, wich can be used for single render pass.

It Supports:

- basic shapes
- images
- 3D models with animation, skinning, node attachment, normal mapping, PBR (not yet)
- gltf loading
- SVG (not yet)
